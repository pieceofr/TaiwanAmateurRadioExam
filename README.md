## 禮包包含
+ 記憶加強法考題測試 APP (本地端開啟 html 使用)
+ 總考題 570 題題庫
+ 六大考題題庫分開的試題以利大家 Divide and Conqure
+ 做題前閱讀筆記 (AI 整理不保證對)
  + 考題-答案-解說 
  + 補充筆記與圖卡
+ 全部 csv 檔可以方便再利用

#### * 這是一個下載 html 網頁與考題到本地端(你的電腦)的程式, 完全沒有聯外, 而且程式碼一目了然(

## 如何使用無線電考試測驗學習 APP
使用者選擇想要**練習的題庫**與**測試數目**後答題.答完題後使用者根據 **EASY/GOOD/HARD/HELL** 給予題目給予題目權重. 選 EASY 表示已學習不會再出現. 權重會影響出現機率
+ 下載
+ 使用
